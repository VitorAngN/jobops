# 02 - Roadmap

## Fase 0 - Fundacao

- Definir escopo do MVP.
- Criar modelagem inicial.
- Definir endpoints.
- Definir layout das telas principais.
- Criar estrutura de monorepo.

## Fase 1 - MVP Back-end

- Configurar Node.js + TypeScript.
- Configurar Express.
- Configurar Prisma.
- Configurar PostgreSQL via Docker Compose.
- Criar entidades principais.
- Criar CRUD de candidaturas.
- Criar filtros basicos.

## Fase 2 - MVP Frontend

- Configurar React + TypeScript.
- Criar layout base.
- Criar tela de dashboard.
- Criar listagem/tabela de vagas.
- Criar formulario de nova vaga.
- Criar filtros por status, area e plataforma.

## Fase 3 - Pipeline

- Criar visualizacao em Kanban.
- Permitir mover candidatura entre status.
- Registrar data de mudanca de status.
- Criar historico de interacoes.

## Fase 4 - Metricas

- Total de candidaturas.
- Taxa de resposta.
- Taxa de entrevistas.
- Vagas por area.
- Vagas por plataforma.
- Tempo medio ate retorno.
- Curriculo com melhor retorno.

## Fase 5 - DevOps

- Dockerfile para API.
- Dockerfile para Web.
- Docker Compose completo.
- GitHub Actions para lint/build/test.
- Deploy em AWS EC2.

## Fase 6 - IA

- Colar descricao da vaga.
- Extrair tecnologias.
- Calcular fit score.
- Sugerir curriculo.
- Gerar mensagem para recrutador.
- Sugerir lacunas de estudo.
